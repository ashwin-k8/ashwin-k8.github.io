SmartCell 3S BMS

A compact, fully-featured battery management system designed for 3-cell lithium-ion packs. Built around the Texas Instruments BQ76920 IC, this module provides complete protection against overcharge, overdischarge, overcurrent, and short-circuit conditions.

FEATURES:

  • •Overcharge protection
  • •Overdischarge protection
  • •Short circuit and overcurrent protection via 10 mΩ shunt resistor on SRP/SRN pins
  • •I2C interface (SDA/SCL) for optional cell voltage monitoring
  • •Dual LED indicator — red while charging, green when fully charged

PIC16F877A Embedded Development Board

A custom-designed PIC16F877A microcontroller development board created using KiCad, featuring integrated power regulation, UART and I2C interfaces, logic-level shifting, reset circuitry, crystal oscillator support, and modular I/O headers for embedded system development.

FEATURES:

  • •Stable 5V regulated power supply using LM7805
  • •I2C interface with bidirectional logic level shifte
  • •Short circuit and overcurrent protection via 10 mΩ shunt resistor on SRP/SRN pins
  • •3.3V peripheral support
  • •MCLR reset circuit with push button
  • •nboard status/debug LEDs
  • •Modular GPIO expansion headers
